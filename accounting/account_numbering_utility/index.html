    Account Number Generation Macro for 5-Level Hierarchical Chart of Accounts¶

    Overview¶

    This VBA macro generates 7-digit account numbers based on a 5-level hierarchical structure in Excel. The format for the account number is as follows:

    • 1 digit for Level 1
    • 2 digits for Level 2
    • 2 digits for Level 3
    • 1 digit for Level 4
    • 1 digit for Level 5

    The account number is padded with zeros to ensure it is always 7 digits long. The account numbers are generated in a hierarchical fashion, starting with Level 1 and progressing through the sub-levels.

    Account Number Structure¶

    The generated account number follows the format:

    • Level 1: 1 digit, followed by 6 zeros.
    • Level 2: 1 digit for Level 1, 2 digits for Level 2, followed by 4 zeros.
    • Level 3: 1 digit for Level 1, 2 digits for Level 2, 2 digits for Level 3, followed by 2 zeros.
    • Level 4: 1 digit for Level 1, 2 digits for Level 2, 2 digits for Level 3, 1 digit for Level 4, followed by 1 zero.
    • Level 5: Fully populated 7-digit account number.

    Example Account Numbers¶

    Level 1 Level 2 Level 3 Level 4 Level 5 Generated Account Number
    1 1000000
    1 1010000
    1 1010100
    1 1010110
    1 1010111

    How It Works¶

    1. Level 1: 1 digit is assigned to represent Level 1, followed by 6 zeros.
    2. Level 2: 1 digit for Level 1, 2 digits for Level 2, followed by 4 zeros.
    3. Level 3: 1 digit for Level 1, 2 digits for Level 2, 2 digits for Level 3, followed by 2 zeros.
    4. Level 4: 1 digit for Level 1, 2 digits for Level 2, 2 digits for Level 3, 1 digit for Level 4, followed by 1 zero.
    5. Level 5: The full 7-digit number is populated with 1 digit for Level 1, 2 digits for Level 2, 2 digits for Level 3, 1 digit for Level 4, and 1 digit for Level 5.

    Using the Macro¶

    1. Open Excel VBA Editor¶

    1. Press ALT + F11 to open the VBA editor.
    2. Go to Insert > Module to add a new module.
    3. Paste the following VBA code into the module.

    2. VBA Code¶

    Sub GenerateAccountNumbers()
    Dim ws As Worksheet
    Dim lastRow As Long
    Dim levelCol As Integer, accNumCol As Integer
    Dim currentLevel As Integer
    Dim i As Long
    Dim level1 As Integer, level2 As Integer, level3 As Integer, level4 As Integer, level5 As Integer
    Dim accNumber As String

    ' Define your worksheet
    Set ws = ThisWorkbook.Sheets("Sheet1") ' Update with your actual sheet name

    ' Define the columns
    levelCol = 3 ' Column where level information is (column C)
    accNumCol = 4 ' Column where account number will be generated (column D)

    ' Get the last row with data
    lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row

    ' Initialize level counters
    level1 = 0: level2 = 0: level3 = 0: level4 = 0: level5 = 0

    ' Loop through all rows
    For i = 2 To lastRow ' Assuming headers are in row 1
        currentLevel = ws.Cells(i, levelCol).Value

        Select Case currentLevel
            Case 0 ' Level 1
                level1 = level1 + 1
                level2 = 0 ' Reset lower levels
                level3 = 0
                level4 = 0
                level5 = 0
                accNumber = Format(level1, "0") & "000000" ' Start with 1 digit for Level 1, followed by 6 zeros
            Case 1 ' Level 2
                level2 = level2 + 1
                level3 = 0 ' Reset lower levels
                level4 = 0
                level5 = 0
                accNumber = Format(level1, "0") & Format(level2, "00") & "0000" ' 1 digit for Level 1, 2 digits for Level 2, 4 zeros
            Case 2 ' Level 3
                level3 = level3 + 1
                level4 = 0 ' Reset lower levels
                level5 = 0
                accNumber = Format(level1, "0") & Format(level2, "00") & Format(level3, "00") & "00" ' 1 digit for Level 1, 2 digits for Level 2 & 3, 2 zeros
            Case 3 ' Level 4
                level4 = level4 + 1
                level5 = 0 ' Reset level 5
                accNumber = Format(level1, "0") & Format(level2, "00") & Format(level3, "00") & Format(level4, "0") & "0" ' 1 digit for Level 1, 2 digits for Level 2 & 3, 1 digit for Level 4, 1 zero
            Case 4 ' Level 5
                level5 = level5 + 1
                accNumber = Format(level1, "0") & Format(level2, "00") & Format(level3, "00") & Format(level4, "0") & Format(level5, "0") ' Full 7 digits
        End Select

        ' Write the account number to the appropriate column
        ws.Cells(i, accNumCol).Value = accNumber
    Next i

    MsgBox "7-digit account numbers generated!"
End Sub

    3. Run the Macro¶

    • Press ALT + F8, select GenerateAccountNumbers, and click Run.
    • The macro will generate account numbers in Column D (or whichever column you specify) based on the hierarchy in Column C.

    4. Adjusting the Sheet Name¶

    If your sheet is named differently, update the following line in the code:

    Set ws = ThisWorkbook.Sheets("Sheet1") ' Change "Sheet1" to your actual sheet name

    Example Input and Output¶

    Input Data:¶

    Level 1 Level 2 Level 3 Level 4 Level 5 Opening Balance
    1 1000
    1 2000
    1 3000
    1 4000
    1 5000

    Output:¶

    Level 1 Level 2 Level 3 Level 4 Level 5 Generated Account Number Opening Balance
    1 1000000 1000
    1 1010000 2000
    1 1010100 3000
    1 1010110 4000
    1 1010111 5000

    Notes¶

    • Make sure your Level 1-5 data is properly structured in the corresponding columns (e.g., Column C) before running the macro.
    • Adjust the column numbers in the VBA code if your data layout is different.
